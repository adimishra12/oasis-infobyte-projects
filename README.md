### 1. Voice Assistant
A simple command-line voice assistant that responds to user commands.

**Features:**
- Greet users with a friendly response
- Tell current time
- Tell current date
- Perform web searches by opening the browser
- Clean user interface with decorative headers
- Professional code structure

**How to Run:**
```bash
python voice_assistant.py
```

**Supported Commands:**
- `hello` - Get a greeting
- `time` - Get the current time
- `date` - Get today's date
- `search <query>` - Search the web for a query
- `exit` - Quit the assistant

### 3. Random Password Generator
A secure random password generator with customizable options.

**Features:**
- Generate passwords of any length
- Choose to include: letters, digits, and/or symbols
- Guaranteed inclusion of at least one character from each selected type
- Input validation for positive length values
- Retry functionality for generating multiple passwords
- Professional error handling
- Clear output with character type summary

**How to Run:**
```bash
python password_generator.py
```

**Example Usage:**
The program will ask you:
1. Enter desired password length
2. Include letters? (y/n)
3. Include digits? (y/n)
4. Include symbols? (y/n)
